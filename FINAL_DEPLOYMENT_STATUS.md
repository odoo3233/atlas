## 🔧 كيفية تشغيل المشروع كاملاً

### 1. تشغيل قاعدة البيانات
```bash
# في Windows
start-database.bat

# أو يدوياً
pg_ctl start -D "C:\Program Files\PostgreSQL\16\data"
```

### 2. تشغيل الخادم الخلفي
```bash
cd backend
npm start
```

### 3. تشغيل الواجهة الأمامية
```bash
cd /workspace
npm run dev
```
