# Atlas Al-Sharq - Bringing Visions to Life

## 🌟 نظرة عامة

Atlas Al-Sharq هي شركة رائدة في تنظيم المعارض والمؤتمرات، تهدف إلى ربط الأعمال بين الصين والمملكة العربية السعودية. يقدم هذا المشروع موقعاً إلكترونياً حديثاً ومتجاوباً يعرض خدمات الشركة ويسهل التواصل مع العملاء.

## ✨ الميزات الرئيسية

### 🎨 تصميم حديث ومتجاوب
- تصميم عصري وجذاب يعمل على جميع الأجهزة
- دعم كامل للغة العربية (RTL)
- تأثيرات بصرية سلسة وتفاعلية
- نظام ألوان متناسق ومهني

### 🌐 دعم متعدد اللغات
- العربية (اللغة الافتراضية)
- الإنجليزية
- الصينية
- تبديل تلقائي للاتجاه (RTL/LTR)

### 📱 واجهة مستخدم متقدمة
- هيدر ثابت مع قائمة تنقل ذكية
- أقسام منظمة ومتجاوبة
- أزرار تفاعلية مع تأثيرات hover
- تصميم glass morphism حديث

### 🔧 وظائف متكاملة
- عرض الخدمات والمنتجات
- نظام طلبات متكامل
- نموذج تواصل مباشر
- نظام باركود للمنتجات
- إدارة المعارض والفعاليات

## 🚀 تشغيل المشروع

### المتطلبات الأساسية
- Node.js 18 أو أحدث
- PostgreSQL
- npm أو yarn

### الطريقة السريعة (موصى بها)
```bash
# انقر مرتين على الملف
RUN_NOW.bat
```

### الطريقة اليدوية
```bash
# 1. تثبيت التبعيات
npm install

# 2. تشغيل الخادم الخلفي
cd backend
npm install
npm start
# الخادم الخلفي يعمل على المنفذ 5001

# 3. تشغيل الواجهة الأمامية (في نافذة جديدة)
npm run dev
```

### العناوين
- **الواجهة الأمامية**: http://localhost:3000
- **الخادم الخلفي**: http://localhost:5001

## 🛠️ حل المشاكل

### مشاكل شائعة وحلولها

#### المشكلة: لا يعمل المشروع
**الحل:**
1. تأكد من تثبيت Node.js
2. تأكد من تشغيل PostgreSQL
3. جرب إعادة تشغيل الكمبيوتر
4. استخدم ملف `RUN_NOW.bat`

#### المشكلة: مشاكل في التصميم
**الحل:**
1. تأكد من تحميل ملفات CSS
2. امسح ذاكرة التخزين المؤقت
3. تحقق من إعدادات المتصفح

#### المشكلة: مشاكل في قاعدة البيانات
**الحل:**
1. تأكد من تشغيل PostgreSQL
2. تحقق من إعدادات الاتصال
3. راجع ملف `start-database.bat`
